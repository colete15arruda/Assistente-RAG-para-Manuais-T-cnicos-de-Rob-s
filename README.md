# Assistente-RAG-para-Manuais-T-cnicos-de-Rob-s
Este projeto implementa um assistente baseado em RAG (Retrieval-Augmented Generation) que permite consultar manuais técnicos e documentação de robótica (PDFs, datasheets, guias ROS2, Arduino/ESP32).   O sistema processa os documentos, cria embeddings, armazena em um vector database e responde perguntas de forma contextualizada usando um LLM.
